### 開発環境の認証について

開発環境は、test-host.xyzというドメインで世界に公開しているので、社内の人間しかアクセスできないように以下の認証を行う。

1. IP制限
2. Basic認証

以下、具体的なやり方。

#### 1 IP制限

指定したIPアドレスからのアクセス以外を弾く、強力な認証方法。これさえつけとけばとりあえず大丈夫、なはず。。。。

やりかた
- Laravelアプリケーションのpublic/.htaccess(backend/web/admin/public/.htaccess)にIP制限の処理を追記  

```
order deny,allow
deny from all 
allow from XXX.XXXX.XXXX.XXXX
allow from YYY.YYYY.YYYY.YYYY
allow from ZZZ.ZZZZ.ZZZZ.ZZZZ
```

(上記の文章を.htaccessの先頭に追記すればOK。allow from 以降は、アクセスを許可したいIPアドレスを記入)  


- IPアドレスは、このページで確認できる。 https://www.luft.co.jp/cgi/htpasswd.php
